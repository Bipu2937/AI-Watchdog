# 🛡️ AI Watchdog: Neural Login Enforcer

A high-fidelity cybersecurity animation and login gatekeeper. This project simulates real-time endpoint protection, malware scanning, and AI-driven behavioral analysis during the login phase.

## 🚀 Features
- **Heuristic Malware Scanning**: Randomized memory address scans.
- **Process Audit**: Real-time PID validation and unauthorized hook detection.
- **Network Isolation**: Procedural port-blocking simulation.
- **AI Validator**: Integrated logic to check login patterns against an AI model.

## 🛠️ Setup
1. Clone the repo: `git clone https://github.com/youruser/ai-watchdog.git`
2. Install dependencies: `npm install`
3. Create a `.env.local` file and add your AI API Key:
   ```env
   NEXT_PUBLIC_AI_API_KEY=your_secret_key_here
   ```
4. Start the engine: `npm run dev`

## 🧪 Usage
- Use any username to see the **Success** path.
- Use the username **hacker** to trigger the **Watchdog Failure** animation.
